# MyWanAndroid
Kotlin写的wanandroid客户端（MVP+Retrofit+OkHttp+RxJava）

### 简介

​	本项目参考了 https://github.com/iceCola7/WanAndroid

​	本项目API见 http://www.wanandroid.com/blog/show/2		

 * 练习kotlin
 * 练习MVP+Retrofit+OkHttp+RxJava
 * 丰富开源库的使用
 * 练习Android Material Design
